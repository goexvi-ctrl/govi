GoVi -- a vi/ex editor for macOS
================================

This disk image contains two programs:

    GoVi.app          the graphical (windowed) editor
    govi              the command-line (terminal) editor

Installing
----------

GoVi.app
    Drag GoVi.app onto the "Applications" folder shown here -- or into any
    folder you prefer.

govi (command line)
    Drag govi onto the "usr-local-bin" folder shown here to install it into
    /usr/local/bin, which is already on your PATH. (Finder may ask for your
    password, since that directory is system-owned.)

    Or copy govi to any directory on your PATH that you control, for example:

        mkdir -p ~/bin && cp /Volumes/GoVi*/govi ~/bin/

    then make sure that directory is on your PATH.

Both programs are self-contained: to uninstall, just delete them.
